# 🌧️ Rainfall Prediction Using Machine Learning

## 📌 Project Overview

This project predicts whether it will rain the next day using historical weather data from Australia. It demonstrates a complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation.

---

## 🎯 Objectives

* Predict rainfall using historical weather data.
* Perform data cleaning and preprocessing.
* Explore weather patterns through visualization.
* Compare multiple machine learning algorithms.

---

## ✨ Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Label Encoding
* Feature Engineering
* Machine Learning Model Training
* Model Evaluation
* Data Visualization
